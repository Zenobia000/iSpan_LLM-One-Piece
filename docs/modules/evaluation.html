核心特色

  • 統一評估接口，支援多種模型
  • 豐富的評估數據集和指標

  • 支援自定義評估任務

包含領域：
  • 人文學科：歷史、哲學、藝術
  • 社會科學：法律、政治、經濟
  • 自然科學：物理、化學、生物

數據工程實踐

數據處理流水線

1

數據收集與整合

從多種來源收集原始數據，包括網頁爬取、API 獲取、公開數據集等

Web Scraping API Datasets
2

數據清洗與預處理

移除噪聲、標準化格式、處理編碼問題、過濾低質量內容

Denoising Encoding Filtering
3

去重與相似性檢測

使用 MinHash、SimHash 等算法檢測和移除重複或高度相似的內容

MinHash SimHash Embedding
4

質量評估與篩選

使用困惑度、分類器評分等方法評估數據質量，篩選高質量數據

Perplexity Classifier Statistics
5

數據標註與驗證

進行人工標註或自動標註，確保標註質量和一致性

Manual Crowdsourcing Validation

實驗手冊

評估與數據工程實驗

Lab 4.1 - OpenCompass 評估實踐

使用 OpenCompass 框架對多個模型進行綜合評估

8小時 中級
  • 安裝配置 OpenCompass 環境
  • 執行 MMLU、GSM8K 等基準測試
  • 分析評估結果和性能報告
  • 自定義評估任務和指標

Lab 4.2 - 數據品質評估與過濾

實踐數據清洗、去重和品質評估的完整流程

6小時 基礎
  • 數據清洗和格式標準化
  • 實現 MinHash 去重算法
  • 使用困惑度評估文本品質
  • 建立數據品質評估指標

Lab 4.3 - 自定義評估基準設計

設計領域特定的評估基準和評估流程

10小時 進階
  • 分析特定領域的評估需求
  • 設計評估任務和數據集
  • 實現自動評估指標
  • 驗證評估基準的有效性
